# CAVE RUNNER — ロードランナー風ブラウザゲーム

`index.html` をブラウザで開くだけで遊べます（ビルド・サーバー不要）。

## 遊び方

- **目的**: 金塊をすべて集めると出口のハシゴが出現。最上段まで登ればステージクリア。
- **操作**:
  - `←→↑↓` / `WASD` — 移動（ハシゴ昇降・ロープぶら下がり）
  - `Z` / `J` — 左足元を掘る、 `X` / `K` — 右足元を掘る
  - `R` — やり直し（残機 -1）、 `Esc` — タイトルへ
- **スマホ**: タッチ端末ではプレイ中に画面下へバーチャルコントローラを自動表示
  （左: 方向パッド〔スライドで方向転換・斜め入力可〕 / 右: 掘るボタン×2・やり直し・タイトル）。
  エディタもタッチで描けます。デスクトップで確認したいときは URL に `#touch` を付けると強制表示。
- 掘った穴は数秒で塞がる。敵を落とせば足止め（踏んで渡れる）、塞がる瞬間に埋めれば倒せる（リスポーンあり）。
- 敵は金塊を拾って持ち歩くことがある。穴に落とすと手放す。

| 1 | はじまりの遺跡 | 基本操作と掘削のチュートリアル |

| 6 | 氷結洞窟 | 地層に埋まった金塊を掘り抜く・高速の敵 |
| 7 | 大空洞 | 浮島づたいの大降下、ロープの切れ目 |
| 8 | 王の宝物庫 | 岩盤の檻に眠る金塊 8 個・敵 5 体の総決算 |

## エディットモード

タイトルの「エディットモード」から。左ドラッグで配置、右ドラッグで消去。

- パレット: 空間 / レンガ / 岩盤 / ハシゴ / バー / 金塊 / 金塊+バー / 落とし穴レンガ / 隠しハシゴ(出口) / プレイヤー / 敵 / 敵+バー
- **テストプレイ** で即プレイ、**保存/読込** はブラウザ (localStorage) に保存
- **書き出し/読み込み** で JSON テキストとして共有可能

## ファイル構成

- `levels.js` — ステージ定義（28×16 の文字マップ）
- `engine.js` — 移動・掘削・敵AI（BFS）・当たり判定
- `render.js` — Canvas 描画
- `game.js` — 画面遷移・入力・メインループ
- `editor.js` — ステージエディタ
- `touch.js` — スマホ用バーチャルコントローラ（タッチ端末のみ有効）

## テスト

```sh
node test/smoke.js     # エンジン動作の回帰テスト（26 項目）
node test/solvable.js  # 全ステージの金塊・出口到達可能性チェック
```
